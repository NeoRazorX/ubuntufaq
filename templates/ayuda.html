La web
Esta es una web de respuestas rápidas para ofrecer soporte a la comunidad Linux, está enfocada en Ubuntu, pero son bienvenidas las preguntas y noticias sobre cualquier otra distribución Linux.
Esta web no tiene ninguna relación con Canonical.
  • Para estar al día de las noticias en torno a Linux tienes las sección actualidad.
  • Si lo que deseas es ponerte en contacto con algún administrador de este sitio web, puedes hacerlo a través de esta dirección de correo electrónico.
Además esta web corre sobre Google App Engine, así que si tienes algún problema ... es cosa de Google!

¿Qué diferencia hay entre esta web y un simple foro sobre Linux?
  • Hay una diferencia clara entre preguntas y noticias (enlaces).
  • Se pueden publicar preguntas, respuestas, enlaces y comentarios de forma anónima, sin necesidad de engorrosos registros.
  • Se hace un seguimiento exhaustivo de las preguntas, desde que se crean hasta que finalmente se marcan como solucionadas, informando al autor de todos los cambios.
  • Cualquier otra persona que no sea el autor también puede disfrutar de este seguimiento.
  • En todo momento se puede acceder al listado de preguntas sin solucionar ordenado por estado.
  • Las respuestas se pueden votar positiva o negativamente, para destacar las mejores y penalizar las que no aporten nada.

¿Qué clase de preguntas puedo hacer?
  • Cualquiera relacionada con Linux, sobre cualquier distribución: Ubuntu, Debian, Linux Mint, SuSE, Fedora...
  • Cuanto más detallada sea la descripción de la pregunta, más fácil y rápido te contestarán. No olvides decir qué versión de Linux estás usando.
  • Las preguntas pueden ser anónimas, pero si inicias sesión podrás hacer un seguimiento más sencillo y completo de tus preguntas, y se te notificarán por email las respuestas.

¿Qué clase de enlaces puedo publicar?
  • Cualquier cosa relacionada con Linux, usa el formulario!
  • Una noticia, un artículo o un blog.
  • Un vídeo de youtube, vimeo o incluso vídeos en formato OGM o WEB.
  • Una imagen o una captura de pantalla, puedes usar el servicio imgur para mayor comodidad.

¿Cómo puedo estar al día de las últimas preguntas y noticias de Ubuntu FAQ sin necesidad de entrar en la web todos los días?
Para eso existen los RSS, solo necesitas un lector como Google Reader. Esta web dispone de dos RSS:
  • El RSS principal que recoge las últimas preguntas y noticias de la web.
  • El RSS de comentarios que recoge las últimas respuestas y comentarios.

Iniciar sesión:
  • Ubuntu FAQ no necesita registro, puedes hacerlo todo de forma anónima, pero si no deseas estar resolviendo captchas o si deseas sacarle más partido a la web (historial, notificaciones, seguimiento de preguntas, etc..) puedes iniciar sesión con tu cuenta Google.
  • Ubuntu FAQ solamente puede acceder a los siguientes datos de tu cuenta Google: nick y email.
  • Ubuntu FAQ no comparte ni compartirá estos datos con nadie más.
  • Si deseas que se eliminen tus datos, envía un email a esta dirección de correo electrónico.

Seguimientos:
Puedes activar el seguimiento automático de una pregunta para no tener que estar pendiente de la página. Con el seguimiento automático se generará una notificación cada vez que alguien responda a esa pregunta.

¿Qué es ese número que aparece a la derecha de cada respuesta?
Esa es la valoración de la respuesta, en función de los votos, sirve para destacar las mejores respuestas y para penalizar las que no aporten nada.

¿Cómo cambio mi avatar?
En esta web usamos el servicio web gravatar para asociar un avatar a una cuenta de correo.
Si deseas cambiar tu avatar entra en http://www.gravatar.com.
  • Si no tienes cuenta:
    • Clic en get your gravatar today.
  • Si ya tienes cuenta:
    • Entra con tu nombre de usuario y contraseña.

¿Qué es ese número que aparece debajo de mi avatar o junto a mi nombre?
Se llama karma ¿Para qué sirve? bueno, eso lo eliges tú:
  • Para motivar a la gente mediante un sistema de status basado en la colaboración.
  • Para nada, pero queda bonito.
  • Para nada y ni siquiera queda bonito.
¿Cómo se calcula? El karma es un valor comprendido entre 0 y 20 (cuanto más mejor), y se calcula de la siguiente forma:
  • 0 < 2respuestas/preguntas < 10
  • 0 < 0.4*enlaces < 4
  • 0 < comentarios/enlaces < 3
  • +1 si está en la portada
  • +1 si está en las últimas respuestas
  • +1 si está en las populares
El sistema actualiza automáticamente el karma de tres usuarios cada 10 minutos, si todavía tienes el tuyo a 0, se paciente, tarde o temprano se actualizará.
